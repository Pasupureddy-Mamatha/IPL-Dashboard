# IPL-Dashboard
Interactive IPL Analysis Dashboard with Power BI/Python
---

This project is an interactive IPL Analysis Dashboard that provides insights into the Indian Premier League from 2008 to 2025. It covers match statistics, player performances, team rankings, and more. The dashboard is designed to help cricket fans, analysts, and enthusiasts explore IPL data effectively.

---

Season-wise Analysis: Select any IPL season to view detailed statistics.

Top Performers: Displays Orange Cap (most runs) and Purple Cap (most wickets) holders.

Team Performance: Shows team points, wins, losses, and rankings.

Aggregated Stats: Total 4’s, 6’s, centuries, and matches played per season.

Interactive Visualizations: Filter and explore data dynamically.
